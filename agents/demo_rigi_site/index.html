The whole integration

One JSON block on orders.create. Done.

Rigi's existing Razorpay checkout stays. Add a transfers array with the sub-merchant IDs and Route handles the rest — split at checkout, per transaction, per ledger.

// Rigi's server, when Priya's fan hits "Renew" —
// one call, four ledgers.
const order = await razorpay.orders.create({
  "amount": 100000,        // ₹1,000 in paise
  "currency": "INR",
  "transfers": [
    { "account": "acc_priya_yoga",   "amount": 78000 }, // 78% → creator
    { "account": "acc_rigi_platform", "amount": 15000 }, // 15% → Rigi
    { "account": "acc_rigi_gst",      "amount":  5000 }, //  5% → GST reserve
    { "account": "acc_rigi_tds",      "amount":  2000 }  //  2% → TDS under 194-O
  ]
})
// That's it. No monthly close. No spreadsheet.
